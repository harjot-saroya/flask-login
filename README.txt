
To run the web server go on your terminal and cd to the location of the file, make sure you have
Pip installed flask on your machine. 

Then once you have reached your file location,
Type in python app.py to run the web server, once it is running it will give you a address to visit
The website.

Paster this address into your browser (it should be "http://127.0.0.1:5000/")
then you may browse the site

I have included dummy data in the sql file for you to use/manipulate
